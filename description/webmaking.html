인터렉티브 웹메이킹

개발자들의 취업시장에서 이력서의 형태가 개성을 담을 링크한줄의 프로필 페이지로 대체되어 가고 있음을 알고 저 또한 개발자다운 자기소개서를 만들어 보고자 마음먹었습니다. 기본적인 HTML, CSS 강의를 마친 후 곧 바로 웹페이지 제작에 착수했지만 웹메이킹은 생각보다 디자인적 역량을 많이 요구하는 작업이었습니다. 레이아웃, 색의 조화 이미지파일의 화질문제등 을 어느정도 해결하고자 무채색 레트로컨셉의 구 버전 페이지를 제작했습니다.
